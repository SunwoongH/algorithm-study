## 📁 Directory Structure
* 이름 폴더 -> 주차별 폴더 -> 소스코드 파일
* 소스코드 파일의 이름은 다음과 같습니다.
  * snake case로 작성하며 ```문제명_이름 이니셜.py``` 입니다.
  * ex) ```reverse_string_hsw.py```

## 📝 Commit Message Rule
* ```이름: n주차 풀이```
* ex) ```sunwoong: 1주차 풀이```
